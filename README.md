# Trust Ticket Project

Trust Ticket은 블록체인 기반의 안전한 티켓 거래 플랫폼입니다. 사용자/주최자용 **모바일 앱**과 관리자용 **웹 포털**로 분리되어 운영됩니다.

---

## 📱 Mobile App (User & Organizer)

사용자와 주최자를 위한 통합 모바일 애플리케이션입니다. **React Native + Expo**로 구축되었습니다.

### 주요 사용자 흐름 (User Flow)
1.  **시작 화면 (Landing Page)**: 앱의 첫 진입점으로, 사용자로 시작할지 주최자로 시작할지 선택합니다.
2.  **통합 인증 (Auth Page)**: 
    - **이메일 방식**: 일반적인 로그인 및 회원가입을 지원합니다.
    - **지갑 방식**: 블록체인 지갑(Wallet)을 통한 인증을 지원합니다. (준비 중)
    - 로그인 성공 시 보유한 역할(Role)에 따라 자동으로 사용자 홈 또는 주최자 대시보드로 이동합니다.
3.  **사용자 메인 (Main / User Home)**:
    - **검색 및 필터**: 상단 검색창과 카테고리 아이콘(콘서트, 스포츠, 전시, 뮤지컬)을 통해 원하는 이벤트를 빠르게 찾을 수 있습니다.
    - **이벤트 탐색**: 진행 중인 이벤트 목록을 카드 형태로 탐색합니다.
4.  **이벤트 상세 (Event Detail)**: 이벤트의 상세 내용과 장소, 일시를 확인하고 직접 티켓을 구매(1차 구매)할 수 있습니다.
5.  **마이페이지 (My Page)**:
    - 개인 프로필 확인 및 로그아웃이 가능합니다.
    - 내 티켓 관리, 리셀 판매 내역 등 주요 메뉴로 이동하는 허브 역할을 합니다.
6.  **내 티켓 관리 (My Tickets)**: 내가 구매하여 보유 중인 티켓 목록을 확인합니다.
7.  **티켓 상세 & QR (Ticket Detail)**:
    - **입장용 QR 코드**: 실시간으로 생성되는 QR 코드를 통해 행사장 입장을 인증합니다.
    - **리셀 판매 등록**: 보유한 티켓에 대해 원하는 가격을 설정하여 리셀 마켓에 판매 등록할 수 있습니다.

### 주최자 주요 기능 (Organizer Features)
- **주최자 대시보드**: 이벤트 등록, 발행, 관리 업무를 수행하는 메인 워크스페이스입니다.
- **이벤트 등록 (Event Create)**: 이벤트명, 상세 설명, 일시, 장소, 가격, 수량을 입력하여 새로운 이벤트를 생성합니다.
- **내 이벤트 관리**: 주최자가 직접 등록한 이벤트들의 판매 현황을 모니터링하고 설정을 변경합니다.

## 💻 Web Application (Admin Portal)

시스템 관리자를 위한 전용 웹 포털입니다. **Vite + React**로 구축되었습니다.

### 관리자 주요 기능 (Admin Flow)
- **관리자 로그인**: 시스템 관리자 전용 인증 화면.
- **관리자 대시보드**: 플랫폼 전체의 통계 및 현황을 한눈에 파악합니다.
- **주최자 승인 관리**: 새로운 주최자 신청 내역을 검토하고 승인하거나 거절합니다.
- **이벤트 감독**: 등록된 모든 이벤트를 모니터링하고 부적절한 이벤트를 관리(Flag)합니다.
- **사용자 관리**: 회원 목록을 조회하고 필요 시 계정을 정지하거나 삭제합니다.
- **거래 / 분쟁 관리**: 리셀 거래 내역을 모니터링하고 분쟁 발생 시 중재합니다.
- **블록체인 모니터링**: 플랫폼에서 발생하는 모든 온체인 트랜잭션 로그를 실시간으로 확인합니다.

## 🏗️ Project Structure
- `/backend`: Spring Boot 기반 API 서버
- `/frontend`:
  - `/mobile`: React Native (Expo) 사용자/주최자 앱
  - (Root): Vite 기반 관리자 웹 포털
- `/blockchain`: Solidity 스마트 컨트랙트 (Foundry 기반)

## 🚀 실행 방법 (Local Development)

### 1. Backend 실행
```bash
cd backend
./gradlew bootRun
```

### 2. 관리자 웹 실행
```bash
cd frontend
npm install
npm run dev
```

### 3. 모바일 앱 실행 (Web View 테스트)
```bash
cd frontend/mobile
npm install
npx expo start --web
```
